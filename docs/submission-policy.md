# 投稿ポリシー（Submission Policy）

## 1. 投稿できる情報の定義

本データベースは**事務所が運営する公式サイトに掲載された情報のみ**を格納します。

### 許可する情報源（ソースURL）

| 種別 | 例 | 備考 |
|------|----|------|
| 事務所公式サイト | `https://stardust.co.jp/idol/name` | **最優先・必須** |
| 公式プロフィールページ | `https://avex.jp/artist/name` | 事務所運営のもののみ |
| 公式グループサイト | `https://groupname.jp` | 事務所管理下のもののみ |

### 許可しない情報源

| 種別 | 理由 |
|------|------|
| 個人ファンサイト・まとめサイト | 正確性を担保できない |
| Wikipedia | 二次情報であり一次ソースではない |
| SNS（Twitter/Instagram/TikTok等） | 本人の投稿でも事務所の管理下でない場合あり |
| ニュースサイト・芸能メディア | 誤報・情報の鮮度問題 |
| 動画サービス（YouTube等） | プロフィール記載情報でも二次的 |

> **原則**: 「事務所が責任を持って管理・更新しているページ」かどうかを基準にしてください。

---

## 2. 投稿できる情報の種類

### アイドル（Idol）

| フィールド | 説明 | 許可基準 |
|-----------|------|---------|
| 芸名（活動名） | 事務所が公表している名前 | 公式サイト記載 |
| 所属グループ | 現在所属しているグループ | 公式発表のみ |
| 所属事務所 | 現在の所属事務所 | 公式発表のみ |
| デビュー年 | 公式デビュー年 | 公式サイト記載 |
| 生年月日 | 公式が公表している場合のみ | 非公表の場合は入力不可 |
| 出身地 | 都道府県レベルのみ、公式公表のみ | 市区町村以下は禁止 |
| 公式SNSリンク | 事務所が公認しているアカウント | 公式サイトからリンクされているもの |
| 公式プロフィール画像URL | 公式サイト上の画像のURLのみ | 画像ファイルの直接アップロードは禁止 |

### グループ（Group）

| フィールド | 許可基準 |
|-----------|---------|
| グループ名 | 公式名称（略称・愛称は別名欄に） |
| 結成日 | 公式発表のみ |
| 解散日 | 公式発表のみ（活動中の場合は空欄） |
| メンバー | 公式サイト記載のメンバーのみ |

### 事務所（Agency）

| フィールド | 許可基準 |
|-----------|---------|
| 事務所名 | 正式名称 |
| 公式サイトURL | 事務所の公式ドメイン |
| 設立年 | 法人登記情報または公式発表 |

### イベント（Event）

| フィールド | 許可基準 |
|-----------|---------|
| イベント名 | 公式発表の正式名称 |
| 開催日時 | 公式発表のみ（変更があれば再投稿） |
| 会場 | 公式発表のみ |
| 出演者 | 公式発表のみ |

## 3. 禁止事項

### 絶対禁止（即座に却下）

- 本名（事務所が公表していない場合）
- 住所・電話番号等の連絡先
- 未確認の噂・憶測・推測情報
- 誹謗中傷・ネガティブな評価
- 著作権保護されたテキストの直接コピー（公式サイトの文章をそのまま転記）
- 卒業・引退後のメンバーを「現役」として投稿
- プライベート画像・無断撮影画像のURL

### 差し戻し対象になる可能性のある投稿

- ソースURLが不明確または事務所公式ではない
- 情報が古い（公式サイトが既に更新済み）
- 略称・愛称を正式名称として入力

## 4. 投稿の流れ

```
投稿者（匿名）
  │
  ├─ メールアドレスを入力（必須）
  ├─ 投稿内容を入力
  └─ ソースURL を入力（必須・1件以上、事務所公式サイトのみ）
        │
        ↓
    [pending: 審査待ち]
        │
   運営による審査（公式サイトとの照合）
        │
   ┌────┼──────────────┐
   ↓    ↓              ↓
approved rejected  needs_revision
（承認） （却下）    （差し戻し）
   │                   │
   ↓               修正後再投稿
本データに反映          │
メール通知送信          ↓
                     [pending: 再審査]
```

---

## 5. ソースURLの要件

- 投稿ごとに**1件以上**のソースURLが必須
- URLは**事務所が管理する公式ドメイン**であること
- URLは投稿審査時点でアクセス可能であること
- URLが後日削除された場合でも、審査時の記録としてデータベースに保持する

---

## 6. 投稿者の責任

投稿時に以下に同意したものとみなします：

1. 投稿する情報が事務所公式サイトに基づいていること
2. 著作権・肖像権・プライバシーを侵害していないこと
3. 虚偽情報でないこと
4. 投稿した情報が CC BY-SA 4.0 ライセンスで公開されることに同意すること
